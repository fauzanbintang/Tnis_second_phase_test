# Tnis_test

Please turn on the less secure apps first for the email notification on this url
https://myaccount.google.com/lesssecureapps

## Server
Make new file `.env` with Template:
```
DB_NAME=
OFFICER_EMAIL=
OFFICER_EMAIL_PASS=
```

Run these commands:
 ```
 $ yarn install
 $ yarn start
 ```

 ## Client
